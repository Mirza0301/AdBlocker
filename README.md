Chrome Ad Blocker Extension


Simple dynamic Chrome extension that blocks ads and trackers. 


Features:
- Blocks common ad networks and trackers out of the box
- Dynamic rule management (add/remove rules via popup)
- Uses Chrome Manifest V3 and declarativeNetRequest API

Files:
- manifest.json
- background.js
- static_rules.json 
- popup.html / popup.js 
- options.html / options.js
